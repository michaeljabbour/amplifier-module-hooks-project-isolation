# Support

## How to get help

If you need help with this module:

1. **Check the README**: Start with the [README.md](README.md) for usage instructions and examples
2. **Search Issues**: Look through [existing issues](../../issues) to see if your question has been answered
3. **Create an Issue**: If you can't find an answer, [create a new issue](../../issues/new)
4. **Amplifier Community**: Join the [Amplifier community discussions](https://github.com/microsoft/amplifier/discussions)

## What to include in your support request

When creating an issue, please include:

- **Module version**: The version of this module you're using
- **Amplifier version**: The version of Amplifier CLI you're using
- **Operating system**: Your OS and version
- **Configuration**: Relevant parts of your bundle configuration
- **Expected behavior**: What you expected to happen
- **Actual behavior**: What actually happened
- **Steps to reproduce**: Clear steps to reproduce the issue
- **Error messages**: Any error messages or logs
